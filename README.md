Projeto do Gonçalo Cruz

# Como usar imagens

Ao inserir uma imagem no GitHub, será possível obter uma referência dela automaticamente para utilizar numa página web.

Exemplo: ao inserir a imagem `A253_43_B.png` emno caminho `intruments/a253`, para obter o link desta imagem, basta construir a URL adicionando a base:
- https://raw.githubusercontent.com/dantovsky/Forgotten-Bagpipes/main

Exemplo de um link completo:
- https://raw.githubusercontent.com/dantovsky/Forgotten-Bagpipes/main/Intruments/a253/A253_43_B.png

Outros formatos possíveos para utilizar:
- https://cdn.jsdelivr.net/gh/dantovsky/Forgotten-Bagpipes@main/Intruments/a253/A253_43_B.png
- https://fastly.jsdelivr.net/gh/dantovsky/Forgotten-Bagpipes@main/Intruments/a253/A253_43_B.png
